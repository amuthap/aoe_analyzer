AOE2 LLM Coach AI - Installation Guide
=======================================

HOW TO INSTALL YOUR GENERATED AI:

1. Generate an AI using the web interface at http://localhost:8080
   - Pick your civ, the AI's civ, and the map
   - Click "Generate AI"

2. Copy the generated .per file to your AOE2:DE AI folder:

   Steam (default):
     C:\Users\<YourName>\Games\Age of Empires 2 DE\<SteamID>\resources\_common\ai\

   Microsoft Store:
     C:\Users\<YourName>\Games\Age of Empires 2 DE\<StoreID>\resources\_common\ai\

3. Also copy the .ai file (same name, different extension) to the same folder.

4. In-game:
   - Start a Skirmish game
   - Click the AI slot dropdown
   - Select your generated AI from the list
   - Play!

NOTES:
- The AI folder may not exist. Create it if needed.
- Each AI needs TWO files: .ai (metadata) and .per (script)
- You can generate different AIs for different matchups
- The LLM customizes build order, unit composition, and aggression based on the matchup
